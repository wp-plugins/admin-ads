Simple plugin to show ads on admin dashboard. Very useful for multi/guest author sites. Don't lose your revenue.

== Description ==

This simple plugin enables you to show ads on your WordPress admin pages. So you can earn more even when you are on admin dashboard.
Very useful if you have a multi author or guest author site. You can add any html code.

= Keep calm and earn more =

= Upcoming features: =
* Hide ads for specific users.
* Show ads in different positions.
* Show different ads in different admin pages.
* Add modal ads
* Earn more even from admin users.
* You can also suggest.


== Installation ==

Upload Admin Ads into you plugin directory (/wp-content/plugins/) and activate the plugin through the 'Plugins' menu in WordPress.

Add your ad code and ad container class (optional) from settings page under settings tab.

Save everything and relax.!!


== Frequently Asked Questions ==

= Which WordPress versions are supported? =
Tested from WordPress versions 3.0 to latest 4.1

= Will ads work all over the dashboard? =
Yes. The plugin will show your ads on head part of all admin pages.

= Should I use the Admin Ads plugin? =
Big Yes!

= Why? =
* If you have a site with many authors and contributors, definitely you can get more ad impressions when ever each users login to admin dashboard. That means you will be earning money without any frontend visitors.!!
